To change the CSS for your optin form, use the selectors below:

Form Container: .opt-bud .email-container {}

Form Image: .opt-bud .email-container .image {}

Form Title: .opt-bud .email-container h3 {}

Form Body: .opt-bud .email-container p {}

Form Button: .opt-bud .email-container .button {}

Form Button Hover: .opt-bud .email-container .button:hover {}

Form Disclaimer: .opt-bud .email-container .disclaimer {}

Form Sucess Message: .opt-bud .email-container.submitted .message {}

Form Spinner: .opt-bud .email-container .spinner {}

Here are some examples:

To increase the font size of the disclaimer: .opt-bud .email-container .disclaimer {font-size:1em;} Note: 1em is the same as 14px

To change the background color of the button to red: .opt-bud .email-container .button {background-color: #ff0000;}

To change the background color of the button to red and the font color to white: .opt-bud .email-container .button {background-color: #ff0000; color: #fff;}

To change the color of the spinner: .opt-bud .email-container .spinner {border-color: rgba(0, 0, 0, .2); border-left-color: #64a74c;}
